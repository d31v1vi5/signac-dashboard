API
===

The complete **signac-dashboard** application interface (API).

Command Line Interface
----------------------

Some core **signac-dashboard** functions are -- in addition to the Python interface -- accessible
directly via the ``$ signac-dashboard`` command.

For more information, please see ``$ signac-dashboard --help``.
